⌨️ Keyboard Shortcuts & Features

Navigation

Backspace Go up one directory
Click folder Open folder
Click file Open file in new tab

Selection

Ctrl/Cmd + Click Select/deselect file
Ctrl/Cmd + A Select all files
Esc Clear selection

Actions

Right-click Show context menu (Download/Delete)
Ctrl/Cmd + K Focus search bar

Features

📤 Upload: Select multiple files to upload to current folder
📁 New Folder: Create a new folder in current directory
🗑️ Delete Selected: Delete multiple selected files at once
🔍 Search: Filter files by name in current directory
🌙 Dark Mode: Toggle between light and dark themes
